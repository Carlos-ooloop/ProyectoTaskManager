# TaskManager Project
## Description:

 Made with FastAPI, Python , MySQL and JWT this is an API Rest when you can add users, make them admin or remove that permissions and in the database you can add task to this users.
 The Tasks are clasified by status and priority. 
 
 ### Stack:
 -Python 3.14.3
 -FastAPI
 -SQLAlchemy
 -MySQL
 -JWT Authentication
 -Pydantic
### Features:
-Register and auth of users
-JWT Access and Refresh Token
-User/Admin roles
-Task and User CRUD
-Soft Delete
-Filters
-Limits on the pages
-Dashboard with the stats
-Logging
